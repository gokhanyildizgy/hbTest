# Test Automation With RobotFramework
Default browser chrome

### INSTALLATION

**IDE:**
Pycharm - Community edition
*plugins:* "Robot Framework Support" and "IntelliBot"

**ENVIRONMENT:**
1. Python v2.x.x
2. Robot Framework

```sh
$ pip install requirements.txt
```
3. Chrome driver

### Run parameter
```sh
$ robot Tests/project.robot
```
